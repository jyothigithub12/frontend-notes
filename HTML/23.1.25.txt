HTML
> Hyper text markup language.
> used to display content in the web page.
> It consists of predefined markup tags which helps in displaying the content. 
        (we cannot create our own tags we can use only predefined tags which helps in displaying the content).

        Tag:

            <tagname>content here...</tagname>

            <tagname= starting Tagname

            </tagname>=ending Tag
